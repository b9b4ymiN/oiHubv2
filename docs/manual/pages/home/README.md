## ใช้เมื่อไร / เหมาะกับใคร
เหมาะกับ 3 สถานการณ์:
1. ผู้ใช้ใหม่ที่ยังไม่รู้ mental model ของระบบ
2. ผู้ใช้เดิมที่ต้องการเข้า dashboard อย่างรวดเร็ว
3. ผู้ใช้ที่ต้องการเข้า tutorial ก่อนเริ่มอ่าน data จริง

## หน้านี้มีอะไรบ้าง
- hero section ที่ย้ำจุดขายของ platform
- feature grid ของ capabilities หลัก
- how-it-works flow แบบ 3 ขั้น
- CTA ไปยัง `/dashboard` และ `/learn`

## ปุ่มและการโต้ตอบสำคัญ
- **Launch Dashboard** → เข้าสู่ cockpit หลัก
- **Interactive Tutorial / Learn More** → ไปหน้าเรียนรู้
- ใช้หน้านี้เพื่อ orient ตัวเอง ไม่ใช่เพื่อ trade execution

## วิธีใช้งานที่แนะนำ
1. ถ้าเพิ่งเข้าระบบครั้งแรก ให้สแกน feature grid เพื่อรู้ว่า platform เน้น OI/derivatives intelligence
2. ถ้ารู้เป้าหมายแล้ว ให้กดเข้า `/dashboard`
3. ถ้ายังไม่คุ้นกับ options/IV analysis ให้เข้า `/learn` ก่อน

## มุมมองแบบ OI Trader
Home ไม่ได้ให้ signal โดยตรง แต่เป็นจุดตั้งต้นของ workflow ทั้งหมด ดังนั้นอย่าตีความข้อความ marketing เป็น promise ของผลลัพธ์การเทรด ให้ใช้เป็น roadmap ว่าหน้าไหนตอบโจทย์อะไร

## ข้อควรระวัง
- ข้อความบนหน้าแรกเป็นระดับ overview; อย่าตัดสินใจเทรดจากหน้านี้
- หากต้องการดูความสามารถจริง ให้ไปหน้า dashboard หรือหน้าวิเคราะห์เฉพาะทาง
