## ใช้เมื่อไร / เหมาะกับใคร
เปิดหน้านี้เมื่อคุณต้องการย้ายจากการเฝ้าหน้าจออย่างเดียว ไปสู่การให้ระบบช่วยแจ้งเตือน setup สำคัญตามสินทรัพย์และ timeframe ที่กำหนด

## หน้านี้มีอะไรบ้าง
- **Quick Start Templates** สำหรับ use case มาตรฐาน เช่น OI Momentum Spike, OI Divergence, Funding Extreme
- **Your Alerts** ตารางรายการ rule ที่สร้างไว้แล้ว
- Actions ต่อแต่ละ rule เช่น enable/disable, edit, delete

## ปุ่มและการโต้ตอบสำคัญ
- **Create Alert** → ไปหน้า `/alerts/new`
- **Template cards** → preload rule บางส่วนให้พร้อมแก้ไข
- **Toggle enabled** → เปิด/ปิด rule โดยไม่ต้องลบ
- **Edit/Delete** → จัดการ rule รายตัว

## วิธีใช้งานที่แนะนำ
1. เริ่มจาก template ถ้า logic ใกล้กับสิ่งที่ต้องการ
2. ตรวจ symbol, interval, severity, และ channel ให้ตรงกับ workflow ของคุณ
3. ใช้หน้า list นี้เพื่อตรวจว่า alert ไหน active อยู่จริง

## มุมมองแบบ OI Trader
alert ที่ดีไม่ใช่ alert ที่ดังบ่อย แต่คือ alert ที่ส่งเฉพาะสถานการณ์ซึ่งคุณพร้อมจะ act ต่อทันที หน้านี้จึงควรใช้เพื่อลด noise และรักษา attention budget

## ข้อควรระวัง
- เปิดหลาย rule ที่ overlap กันมากเกินไปจะทำให้เกิด alert fatigue
- severity ควรสัมพันธ์กับ urgency จริง ไม่ใช่ตั้ง critical ทุก rule
- ตรวจ channel ที่เปิดไว้เสมอ โดยเฉพาะถ้ามี throttling หรือ quiet hours
