## ใช้เมื่อไร / เหมาะกับใคร
เปิดหน้านี้เมื่อคุณไม่ได้ต้องการดู raw panels เยอะ ๆ แต่ต้องการให้ระบบช่วยจัดกรอบคำถาม เช่น ตอนนี้ควรโฟกัสความเสี่ยง, โอกาส, หรือการตัดสินใจส่วนไหนก่อน

## หน้านี้มีอะไรบ้าง
- **Executive Summary** ด้านบน
- tabs: **Signals / Risk / Opportunities / Decisions**
- **SmartQuestionHub** สำหรับถามคำถามตามบริบทของหน้า
- **Quick AI Actions** สำหรับ one-click prompts

## ปุ่มและการโต้ตอบสำคัญ
- **Symbol / Interval selectors** เพื่อเปลี่ยน context หลัก
- **Ask AI** เพื่อส่งบริบททั้งหน้าเข้าระบบสนทนา
- **Quick action cards** เพื่อยิงคำถามมาตรฐาน เช่น market analysis หรือ risk review

## วิธีใช้งานที่แนะนำ
1. ใช้หน้านี้หลังจากคุณอ่าน dashboard รอบแรกแล้ว
2. เปิด tab ให้ตรงกับสิ่งที่ขาด: signal, risk, opportunity, หรือ final decision
3. ใช้ SmartQuestionHub เพื่อเจาะคำถามเฉพาะที่ยังไม่ชัด
4. กลับไป dashboard/orderbook/heatmap ถ้าต้องการตรวจสอบหลักฐานต้นทาง

## มุมมองแบบ OI Trader
Intelligence page คือ **synthesis surface** ไม่ใช่ replacement ของ raw evidence ถ้า AI summary ขัดกับ data layer ที่คุณเห็นใน dashboard ให้เชื่อ evidence ก่อน แล้วค่อยใช้ AI เพื่อช่วยหาสาเหตุว่าทำไมมุมมองต่างกัน

## ข้อควรระวัง
- quick actions ช่วยประหยัดเวลา แต่คำตอบยังควรถูก validate กับข้อมูลจริง
- tab ที่ต่างกันอาจใช้ component ลักษณะใกล้กัน ควรโฟกัสที่คำถาม ไม่ใช่รูปลักษณ์เพียงอย่างเดียว
- ถ้าข้อมูลสดไม่พอ ความฉลาดของ synthesis จะลดลงตาม
